## 🌐 Live Preview

Open the website directly in your browser:

```
index.html
```

No build tools or servers required — just double-click the file or drag it into any modern browser.

### 🖥️ Or serve locally (recommended for best performance)

**Option A — Python (built-in):**
```bash
python3 -m http.server 8080
```
Then open → [http://localhost:8080](http://localhost:8080)

**Option B — Node.js (with live reload):**
```bash
npx serve .
```
Then open → [http://localhost:3000](http://localhost:3000)

**Option C — VS Code Live Server:**
Right-click `index.html` → **Open with Live Server**

## 🎮 Features

- **Interactive 3D Scene** — Procedurally generated Staff of Homa with drag-to-rotate
- **🦋 Butterfly Companions** — 6 animated spirit butterflies orbiting the staff
- **🔥 200-Particle System** — Ember/fire effects with Pyro ↔ Butterfly mode switching
- **📖 Story Scroll Animation** — GSAP + ScrollTrigger reveals across 3 narrative chapters
- **🏮 Dark Gothic Theme** — Crimson red, obsidian black, and gold accents
- **📱 Fully Responsive** — Works on mobile, tablet, and desktop
- **⌨️ Keyboard Shortcuts** — `1` = Pyro Mode, `2` = Butterfly Mode, `Space` = Toggle rotation
- **🌙 Reduced Motion Support** — Respects `prefers-reduced-motion`
